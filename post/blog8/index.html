VS Code最快配置 C 语言开发环境（mac）

a) 准备

  1. 检查 clang

检查所使用的mac是否自带了clang

clang -v

可以直接安装 xcode ，自带gcc

  1. 安装

  • shift+command+X ，并搜索C/C++。

  • 安装Code Runner

  1. 创建一个demo 文件

b）配置

在vs code运行C文件需要配置2个json文件

  • launch.json

  • tasks.json

    1. launch.json 点击打开.c文件，点击IDE上方菜单，Run -> Start Debugging -> 选择 C/C++: clang build active file
    2. tasks.json 点击打开.c文件，快捷键 command+shift+P打开命令行面板，输入tasks:configure task，选择 Tasks:Configure Task -> 选择 C/C++:gcc build active file 如果没有可选项的用户，点击IDE上方菜单，Terminal-> Configure Tasks-> 选择 C/C++:clang build active file ->会自动生成tasks.json文件。
