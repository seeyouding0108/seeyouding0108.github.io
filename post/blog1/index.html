hugo 静态网页创建(准备)

1）hugo 必要的工具

  • homebrew （是一款Mac OS上的软件包管理工具，通过它可以很方便的安装/卸载软件工具等，类似于Linux下的apt-get，node的npm等包管理工具）
  • hugo （Hugo是由Go语言实现的静态网站生成器。简单、易用、高效、易扩展、快速部署。）
  • git （版本控制）

2)安装方法

homebrew：

  • terminal输入 /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install 自动完成安装

hogo 利用braw：

  • brew install hugo

git：

  • brew install git

hugo 网页创建

  • hugo new site myblog生成站点 myblog 可替换

  • 打开网页theme.gohugo.io找主题

  • cd myblog进入path

  • 例如：git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c在themes目录下下载m10c主题

  • hugo server -t m10c --buildDrafts本地启动服务器 在terminal中找到localhost地址，可检查网页是否本地成功打开

  • 写文档：hugo new post/blog.md创建第一个markdown文档

  • 在远端github 创建repository

  • hugo --theme=m10c --baseUrl="http://XXXXXXX.github.io/" --buildDrafts 关联远端地址自动生成public文件

cd public/  
git init //初始化public文件夹使之成为git包
git add .
git commit -m"my hugo first blog"
git remote add origin http://github.com/xxxxxxx/xxxxxxx.github.io.git
git push -u origin master //first time

-更新页面

cd myblog
hugo 
cd public
git add .
git status
git commit -m "add blog post"
git push
